Dependencies

Boost C++

OBJParser relies extensively on Boost Spirit which is a header-only parsing library. It is included as part of the standard Boost collection.

The library is built and tested against Boost v1.60, but any later version should be compatible.

OBJParser solutions expect the Boost library to be located at %BOOST_DIR% environment variable (ie. BOOST_DIR=C:\Boost\boost_1_60_0 such that %BOOST_DIR%\boost\spirit is valid).
