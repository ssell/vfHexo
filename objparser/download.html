Pre-built libraries are available for multiple environments.

For each environment there are four different versions:

  • 32-bit
  • 32-bit with Memory Mapping
  • 64-bit
  • 64-bit with Memory Mapping

If none of the pre-built libraries suit your needs, then the full source code is available.
